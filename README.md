# Carrusel HTML para Google Sites

Este repositorio contiene un **carrusel de imágenes en HTML, CSS y JavaScript puro**,
diseñado para ser **incrustado en Google Sites** mediante código embebido.

---

## Qué hace

- Muestra un carrusel horizontal de imágenes
- Soporta navegación por **swipe** (dispositivos táctiles)
- Permite **redireccionar a una URL personalizada** al hacer clic en cada imagen
- No utiliza frameworks ni librerías externas

---

## Uso principal

Este componente está pensado para:

- Google Sites
- Landing pages simples
- Prototipos rápidos
- Componentes reutilizables sin dependencias

---

## Cómo usarlo

1. Abrir el archivo `index.html`
2. Reemplazar las imágenes por las propias
3. Configurar las URLs de destino
4. Copiar el código dentro de un bloque **Embed / Incrustar** en Google Sites

No requiere instalación ni configuración adicional.

---

## Tecnologías

- HTML

- JavaScript (vanilla)
